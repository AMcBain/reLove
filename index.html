    About

    reLove is a project for talking with http://relive.nu/ and playing back shows listed there. It is mostly feature compatible with the two reLive desktop clients and also offers embedded options for single station or show views.

    Installing

    1a. From ZIP or TAR

    Just use the links at the top of the page and unpack the archive into the desired directory.

    1b. From Git

    If you already have Git or can get it, this is the easiest way to get this project and keep it updated.

    $ cd deployment_parent_directory
$ git clone --recursive https://github.com/AMcBain/reLove.git

    If you're planning to deploy directly to a server this way, make sure your server is configured to block the .git directory and descendants (404 or 403, etc).

    2. Using

    Desktop-local users can just double-click the index.html file in the deployment directory to run this project in a browser.

    If deployed on a server, point your browser to where it was deployed. If that directory is not web accessible you will need to configure your web server. (Google is your friend.)

    2a. Embedding

    This project supports embedding with iframes. This makes it easier to show a single station on a page or include a single show in a blog post. As with step two embedding requires this project be web accessible.

    Styles for the following examples including the recommended minimum sizes.

    iframe.relive.station, iframe.relive.stream {
    width: 500px;
    height: 300px;
}

iframe.relive.stream.player {
    height: 77px;
}

    Single station view. Lists all the shows for the given station and allows users to listen to them.

    <iframe class="relive station" src="/path/to/deploy/directory/embed.html#station-1">
    Your browser does not support iframes.
</iframe>

    Single show view. Restricts the view to only the given show.

    <iframe class="relive stream" src="/path/to/deploy/directory/embed.html#stream-1-9c">
    Your browser does not support iframes.
</iframe>

    If you don't want the segments list or chat log just make the iframe 77px tall, or add the 'player' class in conjunction with the previous styles, and those parts will be hidden.

    These height values are approximate based on the usual default font size for desktop browsers you may need to experiment to get values which work for you.
